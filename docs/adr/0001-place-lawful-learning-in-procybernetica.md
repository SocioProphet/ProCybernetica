# ADR-0001: Place Calibrated Lawful Learning in ProCybernetica

## Status

Proposed.

## Context

Calibrated lawful learning defines a 26-dimensional spectral-unitary-spatial state, calibrated gates, slack-enforced constraints, hyperparameter tuning, observer-stable evidence, and hash-ledger replay.

The model governs adaptive inference rather than merely describing data. It belongs with cybernetic doctrine, conformance law, promotion criteria, evidence handling, and replayable control specifications.

## Decision

The primary repository for calibrated lawful learning is `SocioProphet/ProCybernetica`.

ProCybernetica owns:

- doctrine and conceptual boundary;
- schema contracts;
- conformance requirements;
- reference worked examples;
- validation tests;
- promotion and replay requirements.

Secondary consumers are:

- `SocioProphet/regis-entity-graph` for entity merge governance and prime-topic evidence scoring;
- `SocioProphet/agentplane` for governed bundle execution and replay artifacts;
- `SocioProphet/ontogenesis` for ontology vocabulary and SHACL gates;
- `SocioProphet/socioprophet-standards-storage` for the stabilized normative standard after reconciliation.

## Consequences

This keeps the general theory in the cybernetic doctrine repository while allowing applied repositories to consume stable adapters.

## Non-goals

This ADR does not claim empirical performance. Current examples are illustrative formalizations only. No live data were used.
