| Confidence | Meaning | Default promotion posture |

- preferred label;

- source anchors;
- definitions;
- operational scope;
- allowed uses;
- forbidden uses;
- cross-references;
- implementation surfaces;
- evidence requirements;
- promotion state;
- revision history.
